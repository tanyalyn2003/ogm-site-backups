Calendar Service Account

One-time setup · No staff Google sign-in required

1

Create a Google service account

In Google Cloud Console, create a service account for the schedule app and download a JSON key.

2

Share the OGM calendar

Share the Google Calendar with the service account email and give it Make changes to events permission.

3

Upload the JSON key

Upload the downloaded key file on the server at:

quoter-tool/.data/calendar-service-account.json
4

Reload this page

Once the key is uploaded and the calendar is shared, reload Schedule. The app will connect through the server automatically.
